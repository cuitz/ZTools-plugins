# 开发调试步骤：
1. 启动项目
2. 将“utools打包文件”中的三个文件放到打包后的dist目录下
3. 在uTools开发者工具中，点击“接入开发”


# utools发布步骤：
1. 先打包好项目，打包命令：npm run build
2. 将“utools打包文件”中的三个文件放到打包后的dist目录下
3. 删掉dist中用于调试的.js.map文件和.css.map文件，否则审核不通过


# 更新阿里巴巴图标库
1. 进入图标库项目管理：https://www.iconfont.cn/manage/index
2. 下载至本地，解压后复制css、ttf、woff、woff2 4个文件，进行替换
